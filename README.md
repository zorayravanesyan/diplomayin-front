# Diplomayin — Առողջ ապրելակերպ

Ստատիկ React SPA (առանց backend). Էջեր՝ Գլխավոր, Մեր մասին, Սլայդերներ, Կապ եւ հասցեներ.

## Ճարտարապետություն

- **React 18** + **Vite** + **react-router-dom**
- Բոլոր էջերը կլայենտում; routing՝ `BrowserRouter`
- Ոճեր՝ vanilla CSS (CSS variables, responsive)

## Քայլեր

```bash
npm install
npm run dev    # http://localhost:5173
npm run build  # dist/
npm run preview # preview production build
```

## Էջեր

| Ճանապարհ     | Նկարագրություն        |
|---------------|-------------------------|
| `/`           | Գլխավոր (hero + features) |
| `/about`      | Մեր մասին (բովանդակություն թեզից) |
| `/sliders`    | Սլայդեր (carousel)     |
| `/contact`    | Կապի ձեւ + հասցեներ   |

## Բովանդակություն

Բովանդակությունը հիմնված է դիպլոմային աշխատանքի (առողջ ապրելակերպ, React.js, թվային հարթակներ) թեմայի վրա։ Կոնտակտի ձեւը ստատիկ է — ուղարկումը սիմուլյացվում է (success message), backend չկա։
